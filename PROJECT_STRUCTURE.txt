NLP PROCESSING PIPELINE
========================

1. Input Validation
   ├── Check if blank → return ""
   └── Trim whitespace

2. Model Check
   ├── Models loaded? → Use NLP mode
   └── Models missing? → Use Fallback mode

3. NLP Mode Processing
   ├── Tokenize text into words
   ├── Tag parts of speech (POS)
   ├── Extract keywords by POS tags:
   │   ├── Keep: NN, NNS (nouns)
   │   ├── Keep: NNP, NNPS (proper nouns)
   │   ├── Keep: VBG (gerunds)
   │   ├── Keep: JJ (adjectives before nouns)
   │   ├── Keep: CD (numbers before nouns)
   │   └── Skip: Stop words, action verbs
   ├── Detect compound nouns
   └── Build title

4. Fallback Mode Processing
   ├── Remove sentence starters (I am, I need to, etc.)
   ├── Split into words
   ├── Filter stop words
   └── Build title

5. Title Building
   ├── Take up to 4 words
   ├── Truncate to 40 characters
   ├── Add "..." if truncated
   ├── Capitalize properly
   └── Return result
